# Library-Database-Analysis-SQL-Project
 This project analyzes a library management system using SQL to uncover insights on book availability, borrower activity, and branch-wise loan trends.
# Tools Used
- MySQL
- MySQL Workbench
- SQL
# Database Tables
1. tbl_book
2. tbl_book_authors
3. tbl_book_copies
4. tbl_book_loans
5. tbl_borrower
6.tbl_library_branch
# Key Analysis
- Book availability by branch
- Borrowers with no loans
- Branch-wise loan counts
- Frequent borrowers
- Author-wise book analysis
# Skills Demonstrated
- SQL JOINs
- WHERE clause
- GROUP BY & HAVING
- LEFT JOIN
-  Date filtering
